Detailed Description

This class is a base class for all pure gases used in OpenFCST.

This class contains data provided by its children and implements the methods by means of which different properties of pure gases can be computed.

These methods include:

  • Pressure from the ideal gases, and partial derivatives and
  • Sutherland dynamic viscosity, and partial derivatives
  • Chapman Enskog dynamic viscosity, and partial derivatives where the collision integral is given by
  • Zero bulk viscosity, and partial derivatives
  • Stokes bulk viscosity, and partial derivatives
  • Sutherland thermal conductivity, and partial derivatives where the specific heat capacity at constant pressure is given by and is given by the Sutherland dynamic viscosity formula
  • Chapman Enskog thermal conductivity, and partial derivatives where the collision integral is given by
  • Molar enthalpy, and partial derivatives and where is given by the specific heat capacity at constant pressure formula
  • Water vapor saturation pressure, and partial derivatives

All those methods receive data in SI units. All those methods return the results in SI units.

Warning
Do not create an object of this class. Use its children instead.
Note
For developers: Please update this info appropriately if you add a new method.
Author
Valentin N. Zingan, Chad Balen and Marc Secanell, 2013-16
