Detailed Description

template<int dim>
class FuelCell::Application::AppCathodeKG< dim >

This class solves.

  • steady-state
  • compressible
  • isothermal Kerkhof-Geboers fluid transport equations for a
  • single-phase
  • multi-component case coupled with
  • electron transport equation
  • proton transport equation in a fuel cell cathode.

Valentin N. Zingan
