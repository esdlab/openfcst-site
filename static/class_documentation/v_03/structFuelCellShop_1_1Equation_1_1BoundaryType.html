Detailed Description

This structure describes a boundary type of a derived equation class.

For example:

* set Impermeable walls = 0: no-slip
*

Here

  • boundary_name = Impermeable walls,
  • boundary_id = 0,
  • boundary_condition = no-slip.
Note
Other boundary_condition like Navier slip or perfect slip can be assigned instead.
