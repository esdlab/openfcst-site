Detailed Description

template<int dim>
class FuelCellShop::Layer::SGL24BC< dim >

This class defines a SGL-24-BC MPL, for which effective transport properties are constant.

Warning
This is a template class for different materials. Currently all effective properties are made-up!!!
Author
M. Secanell, 2013
