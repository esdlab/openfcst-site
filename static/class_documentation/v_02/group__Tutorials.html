Detailed Description

This page will list all the tutorials that have developed for FCST.

This is a good starting point for new developers as well as the FCST Trac website which will be coming shortly.

The only tutorial that has been developed at this point is the app_cathode tutorial. This tutorial will describe how a simple application was developed, how it integrates with our contributing libraries and some coding practises that FCST developers adhere to.
